## Rodar o projeto:

### Frontend:

1. Configuração:
   Dentro da pasta `frontend`, renomeie o arquivo `.env.example` para `.env` e configure as variáveis de ambiente conforme necessário.

```
yarn
yarn start
```

### Server:

```
yarn
yarn start:dev
```
